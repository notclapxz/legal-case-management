## 💡 TIPS PARA LA CONVERSACIÓN

### Cómo Hacer las Preguntas

✅ **HAZ**:
- Pregunta una por una, no todas de golpe
- Escucha atentamente (puede mencionar cosas no contempladas)
- Toma notas o graba (con permiso)
- Pregunta "¿por qué?" si algo no queda claro
- Muestra el CSV o documentos para contextualizar

❌ **NO HAGAS**:
- Asumir que sabes las respuestas
- Preguntar todo en 5 minutos (toma tiempo)
- Usar mucho lenguaje técnico
- Presionarlo si no sabe alguna respuesta
